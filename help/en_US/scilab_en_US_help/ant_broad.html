ant_broad

Broad Side Array design

Syntax

[r,xo,twoao,a1,a2,a11,a12,a02]=ant_broad(dB)

Parameters

dB :

In decibels

r :

10^(dB/20)

xo :

Point where T4(xo) = r

twoao,a1,a2 :

Array elements

a11 :

The ratio of the a1 to a1

a12 :

The ratio of the a1 to a2

a02 :

The ratio of the 2ao to a2

Description

Design of a 5 element Broad Side Array which has optimum pattern

Examples

[r,xo,twoao,a1,a2,a11,a12,a02]=ant_broad(20)
